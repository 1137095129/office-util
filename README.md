# office-util
基于easypoi和hutool的word和excel的操作工具
# 项目初衷
之前因公司业务需求，需要生成word表格。由于生成的表格数据存在list集合，且集合数量也不能确定，在网上找了很多工具均无果之后就自己写了一个方法生成word表格。
